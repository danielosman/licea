# mLegitymacja

mLegitymacja to legitymacja szkolna wyświetlana na ekranie telefonu. Dokument można okazywać jak tradycyjną legitymację lub przekazać aplikacji mWeryfikator do weryfikacji danych osobowych i ważności.

## Zabezpieczenia dokumentu

Legitymacja posiada następujące elementy ochronne:
- Hologram ze zmiennym kolorem przy ruchu
- Element dynamiczny (falująca flaga)
- Data wydania
- Termin ważności (wskaźnik kolorowy: zielony/czerwony)
- Znacznik czasowy wydania
- Powtarzający się wzór tła
